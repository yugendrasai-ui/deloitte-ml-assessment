Candidate: Ugendra Sai Ulisetti
College: Vishnu Institute of Technology
Skill Track: AI & Machine Learning

## Project Overview

This project implements a simple machine learning style book recommendation system that suggests books based on genre similarity.

## Features

* Genre-based book recommendation
* Smart title suggestions for incorrect inputs
* Top rated books display
* Interactive command line interface
* Error handling for invalid inputs
* Case-insensitive book search

## Technologies Used

* Python
* Pandas
* CSV dataset for book data

## Project Structure

README.md
main.py
recommender.py
data.csv

## How to Run

1. Install required dependencies (if needed):

   ```
   pip install pandas
   ```
2. Run the program:

   ```
   python main.py
   ```
